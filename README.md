## 🎯 Usage & Setup

```bash
# Clone the repo
git clone https://github.com/shahjalal-labs/dotfiles.git ~/.dotfiles
cd ~/.dotfiles

# Run bootstrap/setup scripts (coming soon) or manually symlink configs:
ln -sf ~/.dotfiles/nvim ~/.config/nvim
ln -sf ~/.dotfiles/tmux.conf ~/.tmux.conf
ln -sf ~/.dotfiles/zshrc ~/.zshrc
# ...and so on

# Install dependencies:
# - Neovim plugins via lazy.nvim or your preferred plugin manager
# - Keyd for keyboard remapping
# - Hyprland and Wayland tools
# - Ranger and SurfingKeys configurations
```

This repo centralizes my personal dotfiles and configurations to enable a seamless, portable, and productive workflow across any Linux machine.  
By sharing it openly, I hope to inspire and help other developers craft their own efficient setups.

## ❓ FAQ & Troubleshooting

- **Q:** How do I update my dotfiles?  
  **A:** Pull the latest changes and re-run your bootstrap or re-symlink configs.

- **Q:** I'm missing some plugins or features after cloning.  
  **A:** Ensure you installed plugin managers and dependencies listed in the Usage & Setup section.

- **Q:** Can I use this setup on non-EndeavourOS or non-Hyprland environments?  
  **A:** Many configs are portable, but some are tailored for my environment. Adjust accordingly.
